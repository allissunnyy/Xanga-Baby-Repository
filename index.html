THE LANGUAGES

Programming Languages I am fluent in and examples of my work.

HTML

Essential Markup Foundation

HTML structures content like text, images, and links, forming the backbone of every web page; it's not a programming language but a required standard for semantic, accessible layouts.

CSS

Styling and Layout Core

CSS controls visuals, layouts (e.g., Flexbox, Grid), animations, and responsiveness; preprocessors like Sass extend it for scalable styles in modern job requirements.

JavaScript

Dynamic Interactivity Standard

JavaScript powers client-side logic, DOM manipulation, events, and APIs; it's the only language browsers natively execute, essential for all front-end roles in all postings.

Python

Core Versatile Language

Python is an interpreted, high-level language known for readability and vast libraries; it powers web back-ends (Django/Flask), data analysis, AI/ML, and automation scripts essential for full-stack roles.
